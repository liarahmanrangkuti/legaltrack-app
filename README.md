# LegalTrack App

LegalTrack adalah aplikasi mobile berbasis Flutter yang membantu pengguna memantau progres pengurusan legalitas usaha secara lebih mudah dan terstruktur.

## Fitur Utama
- Halaman Login
- Input Email dan Password
- Validasi dasar form login
- Tombol Login
- Show/Hide Password
- Link Forgot Password dan Register
- Desain sederhana, responsif, dan sesuai prinsip UI/UX
